1. Set your working directory wherever you want with: setwd().
2. Download the next file in your working directory: https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
3. Unzip the downloaded file.
4. Ensure you have installed dplyr library (if not you can use install.packages("dplyr")).
5. Copy run_analysis.r in your working directory.
6. Load the script with: source("run_analysis.r").
7. Execute the script with: courseProject().
